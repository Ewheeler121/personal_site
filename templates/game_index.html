Controls



Create Mode

Left Click to create, Middle Click to delete, Right Click to set Ending



Connecting Mode

Drag with Left Click to connecting nodes with the type you clicked. Esc to leave connecting mode



Running

hit the 'r' key to run the dnf from start to finish
